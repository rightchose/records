###### 特别长的公式
```
\begin{multline}
x = a + b + c + {} \\
d + e + f + g
\end{multline}
```

###### 公式对齐
```
\begin{aligned}
x = a+b+c+ \\
d+e+f+g
\end{aligned}
```

###### 公式组
```
\begin{gather}
a = b+c+d \\
x = y+z
\end{gather}
\begin{align*}
a &= b+c+d \\
x &= y+z
\end{align*}
```

```

-x,\quad &x\leq 0 \\
x,\quad &x>0
\end{cases}
```
